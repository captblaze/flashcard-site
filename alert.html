ALERT

Definition and Usage

The alert() method displays an alert box with a specified message and an OK button. When this Alert method is used, it will prompt the user to click on an okay button to proceed and it will display the message written in the parameter as an argument.


An alert box is often used if you want to make sure information comes through to the user.


Note: The alert box takes the focus away from the current window, and forces the browser to read the message. Do not overuse this method, as it prevents the user from accessing other parts of the page until the box is closed.

Syntax

alert(message)

Message String Optional. Specifies the text to display in the alert box, or an object converted into a string and displayed, please note that the message must be in a string

Example of an alert

alert("Hello! I am an alert box!!");
